## Проект основан на знаниях, полученных в процессе изучения JavaScript. Мы работали с объектами Document Object Model (DOM), массивами и научились создавать модальные окна на JavaScript. Также мы организовали валидацию форм на сайте, в том числе возможность создания собственной проверки.<br><br>Кроме того, мы применяли свои навыки в работе с API: получали данные от сервера, редактировали и удаляли их, используя fetch-запросы.
